# Weybacks
This script is ready to use on your Kali Linux system for bug bounty reconnaissance!  Find out the Waybacks Urls
Features:\
✅ Input Parameter Support - Interactive prompt for target domain.\
✅ Green Text Output - All URLs displayed in green in terminal.\
✅ Wayback CDX API - Uses the exact API endpoint you provided.\
✅ Line-by-Line Storage - Each URL saved on separate line in text file.\
✅ Error Handling - Validates input, handles API failures gracefully.\
✅ Timestamped Output - Unique filename with timestamp.\
✅ URL Validation - Checks domain format before API request.\
✅ Total Count - Shows how many URLs were discovered.\
✅ Kali Linux Compatible - Tested on Kali, uses standard tools.\
✅ Bug-Free - Proper quoting, error checking, edge case handling.\


## Usage:
# Make executable
chmod +x wayback_extractor.sh

# Run the script
./wayback_extractor.sh

# Enter target domain when prompted
# Example: pipabella.com
